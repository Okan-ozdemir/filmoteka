# Filmoteka

A comprehensive movie discovery application using The Movie Database (TMDB) API.

## Features

- Browse trending movies
- Search movies by title and year
- View detailed movie information
- Watch movie trailers
- Save movies to personal library
- Dark/Light theme support

## Live Demo

Check out the live version: [Filmoteka](https://filmoteka-iota-liard.vercel.app/)

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
```
